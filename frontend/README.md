# Online Appointment Booking Frontend

React + Vite frontend for the Online Appointment Booking Application.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on http://localhost:5173

## Build

```bash
npm run build
```

## Environment Variables

Copy `.env.example` to `.env`:

```bash
cp .env.example .env
```

Make sure the backend is running on http://localhost:3000 before starting the frontend.
