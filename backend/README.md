# Online Appointment Booking API

Express.js REST API with in-memory data storage — no database required.

## Setup

```bash
cd backend
npm install
npm start
```

Server runs on http://localhost:3000

## Development

```bash
npm run dev
```

## Tests

```bash
npm test
```

## API Docs

http://localhost:3000/api-docs

## Environment Variables

Copy `.env.example` to `.env` and fill in values:

```bash
cp .env.example .env
```
